# HLAE-Next-Site

HLAE中文站 —— CS视频制作相关导航与资源集合的中文门户网站。

> this time with Next.js

## 开发

### 技术栈

Next.js(React) + TypeScript + TailwindCSS

### 环境

需要 `nodejs bun`，安装 `node` 后可用 `npm i -g bun` 安装后者。

### 安装依赖

```bash
bun i
```

### 本地开发

```bash
bun dev
```

### 构建

```bash
bun run build
```

### 参与项目

提交 Pull Requests 说明改动的目的。
